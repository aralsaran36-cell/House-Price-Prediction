# 🏠 House Price Prediction

A Machine Learning regression project that predicts **house prices** based on features like location, size, number of rooms, and other housing attributes using the **California Housing dataset**.

---

## 🎯 Problem Statement

Real estate pricing is complex and influenced by many factors. This project builds a regression model to accurately predict house prices — useful for buyers, sellers, and real estate platforms.

---

## 🧠 How It Works

```
Housing dataset (housing.csv)
        ↓
Data Cleaning & Preprocessing
        ↓
Exploratory Data Analysis (EDA)
        ↓
Feature Engineering & Scaling
        ↓
Train regression models
        ↓
Predict house prices
```

---

## ✨ Features

- 🏘️ Predicts house prices based on multiple features
- 📊 EDA — price distribution, correlation heatmap
- 🔧 Feature engineering & normalization
- 📈 Multiple regression models compared
- 🗺️ Location-based price analysis
